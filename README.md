## Tattoo Registration App

This is a simple C# project I made to practice working with desktop applications.
The app is meant for basic tattoo client and session management.

## Features

- Client registration  
- Tattoo session tracking  
- Data management system  
- Clean and simple interface  

## Technologies

- C#  
- .NET  
- Visual Studio  

## Project Structure

- `Program.cs` — entry point  
- `*.csproj` — project config  
- `.sln` — solution file  

## How to Run

1. Open the project in Visual Studio  
2. Build the solution  
3. Run the app  

## About

This project was created as a learning project to improve my C# skills and build a basic management system.
